## Rian's dotfiles

![](https://raw.githubusercontent.com/rian256/dotfiles/main/assets/1.gif)

- Distro: Arch Linux
- Packages: Pacman & yay AUR helper
- Kernel: 6.1.1-arch1-1
- WM: i3-gaps
- Code Editor: NeoVim
- Compositor: Picom
- Panel: Polybar
- System Info: Neofetch
- Audio Visualizer: Cava
- Screen Recorder: VokoScreen
- Gif Screen Recorder: Peek
- Screenshot tools: Flameshot, Gnome-Screenshot (you can choose either of them)
